End-to-End Retail Data Analytics Portfolio Project
Project Overview
This project simulates a corporate-standard data analytics workflow for a leading retail company. It addresses a business problem statement regarding customer shopping behavior, transforming raw data into actionable insights through the full analytics lifecycle: Python processing, SQL analysis, and Power BI visualization.

Key Deliverables
Data Engineering (Python): Performed exploratory data analysis (EDA), category-based median imputation for missing values, and feature engineering (age grouping and purchase frequency mapping).

Database Management (PostgreSQL): Developed a pipeline to migrate processed data from Jupyter Notebooks to a PostgreSQL database using SQLAlchemy.

Advanced SQL Analysis: Executed complex queries utilizing CTEs, window functions (ROW_NUMBER), and CASE statements to perform customer segmentation and ROI analysis on subscription models.

Business Intelligence (Power BI): Engineered an interactive executive dashboard featuring custom DAX measures, dynamic slicers, and trend analysis by demographic and product category.

Stakeholder Communication: Generated a professional project report and a client-ready presentation deck using AI-assisted tools (Gamma).

Tech Stack
Languages: Python (Pandas, NumPy), SQL (PostgreSQL).

Tools: Jupyter Notebook, Power BI, SQL Alchemy, PGAdmin.

Framework: End-to-end corporate workflow (Extract, Transform, Load, Analyze, Visualize).

Insights Summary
Revenue Drivers: Identified young adults as the primary revenue segment.

Shipping Correlation: Validated that express shipping is associated with higher average spend per transaction.

Loyalty Patterns: Segmented the customer base into New, Returning, and Loyal tiers to optimize targeted marketing spend.

Repository Structure
/data: Raw and cleaned datasets.

/notebooks: Python scripts for EDA and ETL.

/sql: Analytical queries for business logic.

/dashboard: Power BI .pbix file and screenshots.

/docs: Project report and presentation PDF.
